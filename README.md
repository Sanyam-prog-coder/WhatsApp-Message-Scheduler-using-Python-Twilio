# 📲 WhatsApp Message Scheduler using Python & Twilio

## 🚀 Overview
This project is a Python-based automation tool that allows users to schedule and send WhatsApp messages at a specific date and time using the Twilio API.

It helps automate reminders, notifications, and personal messages without manual intervention.

---

## 🎯 Features
- 📅 Schedule messages for a future date and time
- 📲 Send WhatsApp messages automatically
- ⏱ Real-time delay execution using Python
- ⚠ Error handling for reliable message delivery
- 👤 Simple user input interface

---

## 🛠 Tech Stack
- **Language:** Python  
- **API:** Twilio API  
- **Libraries:** datetime, time  
- **Platform:** Ubuntu / Linux
